Map Details
This map displays my personally collected and processed orthomosaic drone imagery. All these collections are non-commerical and flown under FAA recreational exception rules.

This is an ongoing personal project to collect, process, and display orthomosaic imagery collected from a drone. Mission planning and execution are conducted using the Dronelink web and mobile applications and all missions are flown using a DJI Mavic Mini SE.

Imagery are processed into orthomosaics using the OpenDroneMap Command Line interface then stored within a Amazon S3 Bucket. Data are served using Python Flask and the pygeoapi. Finally, OpenLayers is used as web display application.
